# Fundið Brand Assets

## Logo Variants

| File | Use |
|------|-----|
| `logo-icon.svg` | Primary icon (favicon, app icon, markers) |
| `logo-wordmark.svg` | Icon + "Fundið" text, dark backgrounds |
| `logo-wordmark-white.svg` | Icon + "Fundið" text, light text for dark backgrounds |
| `logo-icon-mono-dark.svg` | Ink-colored icon for print/monochrome |
| `logo-icon-mono-light.svg` | White icon for dark backgrounds |

## Colors

| Token | Hex | Use |
|-------|-----|-----|
| Amber (brand) | `#E08A50` | Logo, primary actions, CTAs |
| Amber dark | `#C87640` | Hover/active states |
| Amber light | `#FBF0E6` | Tinted backgrounds |
| Ink | `#2C2520` | Body text, headings |
| Ink light | `#5C534D` | Secondary text |
| Muted | `#9C8B7E` | Captions, placeholders |
| Border | `#E8E0D8` | Dividers, card borders |
| Surface | `#FAF7F4` | Page background |
| Lost | `#D9534F` | Lost item badges |
| Found | `#4A9B6A` | Found item badges |

## Typography

- **Primary**: Inter (400, 500, 600, 700)
- **Fallback**: system-ui, -apple-system, sans-serif

## Logo Concept

Map pin + checkmark. "Fundið" means "found" in Icelandic — the checkmark represents the moment of reunion between owner and lost item.
